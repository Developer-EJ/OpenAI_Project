# Jungle Campus

## 프로젝트 소개

Jungle Campus는 교육 과정 구성원이 같은 가상 캠퍼스에 접속해 실시간으로 만나고, 대화하고, 파티를 모집할 수 있는 2D 커뮤니티 서비스입니다.

초기 버전에서는 하나의 맵 안에서 만나고 채팅하는 기능만 제공했습니다. 현재 버전에서는 영역 이동 UI와 파티 시스템을 추가해, 사용자가 공간을 오가며 더 자연스럽게 모이고 함께 활동할 수 있도록 확장했습니다.

## 주요 기능

- 3개의 교육 과정 중 하나를 선택하고 바로 입장할 수 있습니다.
- 같은 공간의 사용자 위치와 움직임이 실시간으로 동기화됩니다.
- 전체 채팅과 근거리 채팅으로 상황에 맞게 대화할 수 있습니다.
- 로비를 제외한 공간에서 파티를 만들고, 정해진 인원만큼 참여할 수 있습니다.
- 같은 area에 있는 사용자끼리 음성 연결을 시도할 수 있습니다.

## 기존 버전 대비 개선점

- 단일 맵 기반 구조에서 로비와 세부 공간으로 이동하는 구조로 확장했습니다.
- 단순 채팅 중심 서비스에서 파티 모집과 참여 중심 서비스로 발전시켰습니다.
- 사용자 간 상호작용 범위를 `hall`이 아닌 `area` 기준으로 정리해 같은 공간 경험을 강화했습니다.
- 음성 연결 상태를 확인할 수 있는 디버그 UI를 추가해 문제 지점을 빠르게 파악할 수 있게 했습니다.

## 테스트 및 검증

### 테스트 케이스

0. 입장 및 사용자 동기화
- 다른 사용자가 참가하면 맵에 상대방이 정상적으로 표시되고, 움직임도 실시간으로 보이는 것을 확인했습니다.

1. 같은 area 내 음성 대화 기능
- 같은 area에 있는 사용자를 감지하는 데는 성공했습니다.
- 하지만 송신한 음성을 상대방이 수신하는 단계는 실패했습니다.

2. 파티 시스템
- 파티 생성 시 설정한 최대 인원수만큼만 참가가 가능한 것을 확인했습니다.
